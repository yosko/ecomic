  • Type '1' - '6' to show the designated comic (if there are that many). Default testscene.js only includes 2 sample comics.
  • Type 'X' to take a screenshot and turn the canvas into an image, opened in another tab/window.
  • Type 'Q' to turn off backgrounds.
  • Type 'W' to turn off drop shadows.
  • Feel free to modify the JSON file loaded into the text area. Press the button "load Comic JSON" to reload it and rebuild all the comics.
